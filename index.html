What is Zone.js?

Stand-alone ES5 library (~500 LOC)

Implements a new primative – Zone – for browsers

Zones aren't new

  • Zones in Dart
  • Domains in Node
  • Async Listeners in Node

Zone.js is a generalized solution; lets you compose behaviors

There's also zone (npm module), which implements zones for node.

  3. Profiling
  4. And More

Demo

A Zone is a behavior for a certain execution context

Zone.js ships with these zones:

  • Counting Zone – lets you know when all async tasks are done
  • Long Stack Trace Zone – keeps track of the call stack across async tasks

How does Angular use Zones?

At the end of each task, we detect changes to observed objects. This is how we know whether or not to re-render a view.

If you've used Angular...

In AngularJS 1.x, you had to call $rootScope.$apply if you changed some model outside of Angular's lifecycle. Now you won't have to!
